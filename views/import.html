This form accepts a CSV file which should have the following headings: title, descr, start date, start time, end date, end time, address or location and tags. Tags are comma-separated. Categories is currently accepted as a synonym for tags. Date and time are accepted as synonyms for start date and start time. All columns are optional. The descr column is plain text. Any additional headings are politely ignored. Please note that duplicates are not detected. If you import duplicates you may need to remove them manually.
